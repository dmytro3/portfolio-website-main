What I can do.

Test Automation Framework & Development

Design, setup, and maintenance of customized test automation frameworks integrated with tools like Selenium, Appium, Cucumber, TestNG and JUnit. This encompasses both the foundation of the framework and the creation of automated functional, regression, and smoke tests. Integration into CI/CD pipelines ensures seamless execution in development cycles. Api and Db automation with Rest Assured, JDBC and MongoDb libraries.

Functional, Front-end & Back-end Testing

Deliver a comprehensive testing process that covers a wide array of methodologies, including black-box, grey-box, functional, integration, end-to-end, regression, sanity, and smoke. Expertise extends to both the user-facing aspects (front-end) and the underlying server-side components (back-end), including thorough testing of APIs and databases, both relational (MySQL, PostgreSQL) and non-relational (MongoDB).

Performance & Load Testing

Identify bottlenecks and ensure scalability by designing and executing performance tests. Tools like Postman, RestAssured, Jackson, JMeter and BlazeMeter are utilized to guarantee that applications can handle anticipated user loads and respond efficiently.
